This package is release version of RBET by Lei Sha, Microsoft Research Asia

The RBET release version is developed in Netbeans IDE.

rbet_release.RBET_release is the main class


1, Test

RBET_release -test ../props/ace11chunker.properties testfilelist testfiledir outputfiledir


2, Train

RBET_release -train ../props/ace11chunker.properties trainfilelist trainfiledir Modelfiledir

An simple example is in the ./example folder
